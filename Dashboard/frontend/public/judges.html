Task Weave — Submission Notes

How to install & test this entry

This page is hosted on the live app itself — the dashboard you'll sign into in step 2 lives at this same site's homepage.

Why a Drive link, not the Chrome Web Store

Task Weave is a Chrome extension paired with a web dashboard. Normally a Chrome extension gets distributed through the Chrome Web Store — but Google's store listing review can take anywhere from a few days to a couple of weeks, which doesn't fit inside a hackathon judging window. So instead of a store link, this submission ships as a pre-built, ready-to-load extension package via Google Drive, installed the same way any developer loads an in-progress extension locally: Developer mode → Load unpacked.

This is the standard, Google-documented way to run an extension that hasn't been published yet — nothing here requires bypassing browser security, just enabling the one developer toggle Chrome ships for exactly this case.

1. Download the extension

Download from Google Drive

Unzip it once downloaded — you should get a folder containing manifest.json and a handful of subfolders (popup, focus-lock, task-capture, etc.).

2. Load it into Chrome

  1. Open chrome://extensions in the address bar.
  2. Toggle Developer mode on (top-right corner).
  3. Click Load unpacked.
  4. Select the unzipped folder.
  5. "Task Weave" should now appear in your extensions list and toolbar.
Sign in to unlock task & focus features

Click the extension icon → Dashboard, which opens the live web app and signs you in with Google. Once signed in there, the extension picks up the same session automatically — no separate login inside the popup.

3. What to try

  • AI planning chat — on the dashboard, type something like "exam tomorrow, need to revise DSA and DBMS" and watch it get broken into tasks with a next micro-step each.
  • Task capture — on any webpage, right-click → "Add to Task Weave," or press ⌘⇧Y (Mac) / Ctrl+Shift+Y (Windows) to capture the page as a task. Works without signing in.
  • Focus timer + task picker — open the extension popup, pick a task (🎯 button) or type a custom focus, hit Start.
  • Distraction blocking — with a focus session running, try visiting YouTube/Instagram/X/Reddit — it redirects to a lock screen until the session ends. Works without signing in.
  • Workflows — on the dashboard's Workflows page, describe a recurring procedure in plain English and watch AI turn it into an automation.
One thing to know

This points at our live backend and database, not a sandbox — anything you create (tasks, sessions, goals) is real data in our production Firestore. Fine for a demo, just not isolated per-judge.
